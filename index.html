세션 행동 요약

15개 녹화 세션 분석 결과, 대부분의 사용자가 6~12초 내에 1회 클릭 후 이탈하는 패턴을 보입니다. 적극적으로 탐색한 사용자(01:21, 11페이지)는 단 1명이었으며, 즉시 이탈(1�4�, 0클릭) 세션도 다수 존재합니다. 이는 이벤트 페이지가 사용자에게 다음 행동을 명확하게 유도하지 못하고 있음을 시사합니다.

Design Insights & Action Items
디자인 개선 제안
  • 1

    히어로 영역에 강력한 CTA 배치

    현재 사용자의 100%가 페이지를 끝까지 스크롤하지만, 클릭 전환이 거의 없습니다. 스크롤 없이 보이는 첫 화면(Above the Fold)에 "지금 이벤트 참여하기" 같은 명확한 CTA 버튼을 크게 배칙하세요. 버튼 색상은 주변과 대비되는 색으로, 최소 48px 높이를 확보해주세요.

    예상 효과: Quick Back율 감소, 클릭 전환율 증가
  • 2

    LCP 6.1초 → 2.5�4� 이하로 개선

    히어로 이미지 또는 주요 비주얼 에셉이 로딩에 6초 이상 걸리고 있습니다. 이미지를 WebP로 변환하고, 히어로 이미지에 preload를 적용하세요. Lazy loading은 ATF(Above the Fold) 밖의 이미지에만 적용해야 합니다. CLS 0.4 해결을 위해 이미지/영상 영역에 반드시 width/height 또는 aspect-ratio를 지정하세요.

    예상 효과: 이탈률 감소, SEO 점수 향상, Core Web Vitals 통과
  • 3

    이벤트 → 채용정보 전환 동선 강화

    가장 활발한 사용자의 여정이 이벤트 → 메인 → 채용정보였습니다. 이벤트 페이지 하단에 "수능끝! 지역별 알바 바로 찾기" 같을 연계 세션을 추가하며, 메인을 거치지 않고도 바로 채용정보로 유도할 수 있습니다.

    예상 효과: 세션당 페이지 수 증가(현재 2.14), 채용공고 조횈 증가
  • 4

    외부 유입 채널 확띀

    현재 유입의 94.7% 가 알바몬 내부(m.albamon.com)입니다. 네이버 블로그 1건 외 외부 유입이 전무합니다. 인스타그램, 카카오톡 공유, 네이버 검색 광고 등 외부 채널 마케팅을 병행하면 도달 범위를 크게 늘릴 수 있습니다.

    예상 효과: 신규 사용자 도달 범위 확대, 이벤트 인지도 상승
  • 5

    모바일 터치 타겟 및 인터랡션 한적화

    사용자의 90% 이상이 모바일(iOS 중심)로 접속합니다. 모든 터치 타겟을 최소 44x44px로 확보하고, 스크롤 중 나타나는 Floating CTA 버튼을 추가하면 어느 위치에서든 전환이 가능합니다. iOS Safari에서의 렌더링 테스트도 반드시 수행하세요.

    예상 효과: 모바일 전환율 향상, 인터랡션 품질 개선
  • 6

    이벤트 참여/공유 퍼널 재설계

    깔때기(Funnel) 분석에서 "refer friends" 전환율이 0%입니다. 이벤트 참여 → 친구 공유의 흐름이 사용자에게 발견되지 않거나, 참여 허들이 너무 높을 수 있습니다. 공유 버튼을 스크롤 고정(Sticky)으로 노출하고, "친구에게 공유하면 OO 혜택" 같은 인센티브를 시각적으로 강조하세요.

    예상 효과: 공유 전환율 0% → 목표 5% 이상
